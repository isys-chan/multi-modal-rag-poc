# Specification 
## 概要 
- Multi-modal RAG AIシステムの構築 
- テキスト、画像、音声などの複数のモーダルリソースを統合したAIシステム 
- 検索と生成の両方の機能を持つ 
- ユーザーが自然な言語で質問し、多様な形式の情報を提供できる

## 機能 
- 新規テキスト、画像のベクトル化（pdf, png, txt） 
- モーダル間の関連性を考慮した検索と生成 - ユーザーインターフェースを通じて自然な言語での操作 
- ユーザー入力とシステム応答のログ管理 
- (注)PoCのため、音声入力は実装しない。必要最低限で実装する。

## 実装 
### フロントエンド 
- Streamlitを使用（PoCなので必要最低限） 
### バックエンド 
- Python LangCHain, LangGraphで実装
- 環境変数は.envに格納済み 
- load_dotenv()で読み込み 
- Logging 
- Pythonのloggingモジュールを使用
- ユーザーの質問とシステムの回答をログに記録 
### データベース 
- Chromaを使用 
- ベイカレ社内文書、一般公開資料を格納（後でアップロードする） 
### LLM 
- OpenAI GPT-4Vを使用
### フォルダ構成
- PoCなのでとにかくシンプルに最小限

## 最終成果物 
- Multi-modal RAG AIチャットシステムの動作するプロトタイプ 
- 画像・テキストをアップロードできるインターフェース
